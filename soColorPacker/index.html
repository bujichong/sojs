基于jQuery的网页拾色器：soColorPacker 1.0 (点击下载)

所有参数：

$(obj).soColorPacker({
	changeTarget:null, //颜色选择后填入或改变的目标对象，默认null为当前引用方法对象
	textChange:true, //是否将颜色值填入target对象
	colorChange:1, //值：0,1,2 ，0：不改变，1：改变文字颜色，2：改变背景颜色
	selfBgChange:false,//改变自身背景颜色
	size:2, //值：1,2,3，改变拾色器表现大小，1：小，2：中，3：大
	x:0,////拾色器坐标x轴偏移量
	y:20, //拾色器坐标y轴偏移量
	styleClass:null,//自定义添加class名，不能为默认的 "colorPackerBox"
	callback:callback:function(){}//点击颜色单元返回事件
});

未采用生僻的jquery特性，能兼容jquery1.2.6以上所有版本；
min版2.5k，兼容所有主流浏览器，如有发现问题或有更好的改进方案，请反馈给我，十分感谢 :)

2- 颜色选择框选色

$('#img_colorPack_2').soColorPacker({
// 图片为操作对象
	changeTarget:'#txt_pack_2',// 目标对象为文本框#txt_pack_2
	selfBgChange:true// 开启自身背景改变属性
});
//通过指定target对象和开启自身背景改变属性，我们可以将颜色赋值对象和引用方法对象分开显示

3- 改变对象背景色，使用不同尺寸的拾色器

$('#img_colorPack_3').soColorPacker({
	size:3',// 使用大尺寸拾色器
	colorChange:2// 颜色改变模式选择2，为改变背景色
});
//可以通过size值为1到3选择适合实际情况的选择器大小，colorChange值0到2来选择改变对象的颜色属性

5-自定义拾色器样式

$('#txt_pack_5').soColorPacker({
	styleClass:'colorPick_my'// 自定义添加样式类名
});
//通过添加自定义类名，可以自定义单元宽度来改变选择器大小以及对应dom对象样式，对样式更熟悉一些的开发者可以更灵活的定义
//调用bgIframe，解决ie6下select层级最高的bug，如果不需要兼容ie6，可不调用jquery.bgiframe-2.1.1.js

6-点击颜色单元返回事件

$('#txt_pack_6').soColorPacker({
	callback:function (c) {//自定义点选颜色的返回事件
		alert('你选择的颜色是：'+c.color);
	}
});
